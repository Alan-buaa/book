# k8s接口处理说明

## 1. 提交阶段

| 返回状态码 | 界面状态 | 说明                                                       |
| ---------- | -------- | ---------------------------------------------------------- |
| 200        | 提交成功 | 提交成功，继续下一阶段                                     |
| 400        | 提交失败 | StatusBadRequest，一般为参数错误，需记录对应消息供定位使用 |
| 401        | 提交失败 |                                                            |

*上表中状态码，go sdk net.http status.go中有罗列*
